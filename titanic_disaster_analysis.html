Titanic Disaster Analysis¶

The sinking of the Titanic is one of the most infamous shipwrecks in history.

On 15th april 1912, the widely considered “unsinkable” Titanic sank after colliding with an iceberg. Unfortunately, there weren’t enough lifeboats for everyone on board, resulting in the death of passengers and crew members. While there was some element of luck involved in surviving, it seems some groups of people were more likely to survive than others. In this Datasets, we will try to build a predictive model which help to understand and analysis of this Disaster.

def impute_age(cols):
    Age = cols[0]
    Pclass = cols[1]
    if pd.isnull(Age):
        if Pclass == 1:
            return 37
        elif Pclass == 2:
            return 29
        else:
            return 24
    else:
        return Age
